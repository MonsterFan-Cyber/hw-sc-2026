需求，使用 web 架构，对 docs/spec/snapshoot_format.md 当中的  .snpst  文件进行解析，并进行动画播放

## 一、要求

1） 选择.snpst文件按钮和功能
2） 播放按钮，暂停按钮，以及功能
3） 选择开始帧按钮，旁边有输入开始帧数字的填空框，默认是暂停状态
5） 帧导航条，可以手动拖动进度条来控制进度，右上角第4行，上面的数字
6） 求解日志区域
7） 绘图框，除了右边的控制按钮区域，其他地方都是绘图区域
8） 需要支持暗色主题和白色主题
8） 除了上面需求，你可以自由发挥，根据 docs/spec/snapshoot_format.md 数据特点来设计WebUI界面
